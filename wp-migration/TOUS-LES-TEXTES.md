# TOUS LES TEXTES DU SITE NSAD
## Fichier de référence pour les éditeurs non-techniques

> **Comment utiliser ce fichier :**
> - Ce document liste tous les textes importants du site, page par page
> - Si vous souhaitez modifier un texte, repérez-le ici, notez la modification souhaitée
> - Transmettez ce fichier annoté à votre développeur, OU
> - Retrouvez le texte correspondant dans Elementor et modifiez-le directement
>
> **Légende :**
> - `[TITRE PRINCIPAL]` = Le grand titre en haut de la page (H1)
> - `[TITRE SECTION]` = Titre d'une section (H2/H3)
> - `[TEXTE]` = Paragraphe de contenu
> - `[BOUTON]` = Texte d'un bouton
> - `[TEL]` = Numéro de téléphone
> - `[EMAIL]` = Adresse email

## NOTES POUR LES ÉDITEURS

### Ce que vous pouvez changer librement
- Tous les textes listés ci-dessus
- Les photos (format JPG ou WebP recommandé)
- Les témoignages
- Les actualités et articles
- Les horaires et coordonnées
- Les textes des FAQ
- Les listes de partenaires

### Ce qu'il ne faut pas changer sans développeur
- Les titres de pages dans l'URL (ex: `/nous-contacter/`) — cela casse le référencement
- La structure des menus de navigation
- Les couleurs et polices du site
- Les formulaires de contact
